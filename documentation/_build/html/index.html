class DAC8568.DAC8568(baud_rate: int)¶

Class to interface with the DAC8568 8 channel digital-to-analog converter via the FT232H controller.

clear_all_to_mid() → None¶

Sets all output registers to half way between 0 and maximum

clear_all_to_zero() → None¶

Sets all outputs registers to 0

internal_reference_on() → None¶

Turns the internal reference on (2.5V). There should be not external reference connected when this occurs or it may damage the board

static to_bytes(word: int) → bytearray¶

Converts an integer to an array of 8-bit bytes with length 4 that can be processed by busio.SPI write for the Ti DA8568

Parameters

word – Integer with 32 bits. Raises assertion error when more significant bits are present

Returns

A length 4 array of 8-bits bytes that represent the value of word
