Power supply module¶

class powerSource.DAC8568(baud_rate: int)¶

Class to interface with the DAC8568 8 channel digital-to-analog converter via the FT232H controller.

Initializes the connection with the microcontroller board. Takes control of pins D0, D1, D3 on the board.

Parameters

baud_rate – Rate that data is sent from the controller to the DAC. The maximum rate for the DAC is 50MHz but the interface board can’t get that fast.

import numpy as np
from code.powerSource import DAC8568

def generate_points():
    n = 1000
    t = np.arange(n)
    y = np.sin(t/n*2*np.pi)/2. + 0.5

    return t, y

def smooth_point():
    _, Y = generate_points()

    dac = DAC8568(baud_rate=100000)
    while True:
        for y in Y:
            dac.write_and_update(channel="A", value=float(y))

if __name__=='__main__':
    smooth_point()
