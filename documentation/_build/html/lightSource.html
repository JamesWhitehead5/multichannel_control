Light source module¶

class lightSource.TSL510¶

This class provides a simple interface with the TSL-510 Laser. It creates an object that consists of setters and getters. Most methods require the Laser Diode to be turned on in order to work. After turning the power on or the LD output off, it is recommended that you wait at least 30 seconds before turning on LD output. While the LD light is blinking, communication is not possible.

Constructor finds and connects to the VISA resource with name in ID string Prints confirmation to STDOUT. Raises exception if cannot connect :raises: RuntimeError

Uses power meter inside of laser to measure actual power output (at source)

Returns

Measured power in system units (dBm or mW)

set_color(color: float) → None¶

Sets the color of light coming out of the laser.

Parameters

color – The color of light coming out of the laser. Units are either nanometers or THz, depending on what has been set beforehand.

set_power(power: float) → None¶

Sets laser power in units set beforehand

Parameters

power – Output laser power in either dBm or mW.

set_power_unit(value: bool) → None¶

Sets the power unit for the laser. Power field can have units dBm (dBmW) or just mW.

Parameters

value – If false, the power units are dBm. If true, mW.

set_wavelength_unit(unit: bool) → None¶

Wavelength field can either represent the wavelength(in nm) or the frequency(in THz). Parameter num can either be 0 or 1.(0:nm, 1:THz) Raises exception if num is out of range

Parameters

unit – If unit is false, the unit is set to nanometers. If true, TeraHertz
